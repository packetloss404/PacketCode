Six core capabilities designed to keep you in flow.

Multi-Pane Sessions

Run Claude Code and Codex CLI side-by-side in real PTY terminals. Split, resize, and monitor in real time.

Kanban Issue Tracker

Drag-and-drop board with priorities, labels, epics, and acceptance criteria. Link issues directly to AI sessions.

Flight Deck

Autonomous orchestration layer. Create flights, link issues, and let agents build while you review. Status rolls up automatically.

Agent Profiles

Switch between 5 built-in personalities or create your own. Each injects custom prompts and model defaults.

GitHub Integration

Connect repos, browse issues, create PRs, and import tickets — without leaving the cockpit.

Deploy Pipeline

Configure and trigger deployments from a single panel. Run history, environment variables, live terminal output.
