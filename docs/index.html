Features

Everything you need.
Nothing you don't.

A full-featured IDE experience designed from the ground up for AI-first development workflows.

Multi-Pane AI Sessions

Run Claude Code and Codex CLI side-by-side in full PTY terminals with real-time status monitoring. Split, resize, and manage as many sessions as you need.

Kanban Issue Tracker

Built-in board with drag-and-drop, priorities, labels, epics, and acceptance criteria. Link issues directly to AI sessions.

Insights Chat

Conversational Q&A against your full codebase. Ask about architecture, find bugs, or explore patterns.

Code Quality

On-demand quality analysis with AI-powered feedback on security, patterns, and best practices.

Spec2Tick

Paste any project spec and let Claude parse it into structured kanban tickets automatically.

MCP Server Hub

Manage Claude Code's MCP server configs — add, edit, and toggle servers across global and project scopes.

Project Scaffolding

New Project wizard with 6 templates — Next.js, React+Vite, FastAPI, Rust CLI, Express, and Blank.

Deploy Pipeline

One-click deploy with live terminal output. Auto-detects configs from package.json, Vercel, Netlify, and Docker. Run history tracks every deployment with status and duration.

Agent Profiles

Give your AI a personality

Preconfigured agent personalities that change how your sessions behave. Each profile injects a system prompt, sets a default model, and gives visual identity.

  • ✓ 5 built-in profiles tuned for different workflows
  • ✓ Create unlimited custom profiles
  • ✓ Quick-switch from the toolbar without opening settings
  • ✓ Profiles auto-fill model selection in new sessions

Module System

Modular by design

Tools are registered declaratively through a manifest system. Enable or disable modules from Settings — the menu, routes, and keyboard shortcuts update automatically.

  • ✓ Each tool is a self-contained module with icon, category, and component
  • ✓ Toggle modules on/off without restarting — state persists across sessions
  • ✓ Disabled modules are removed from the Tools menu and keyboard shortcuts
  • ✓ Adding a new tool is a single manifest file — no more editing 4+ files
